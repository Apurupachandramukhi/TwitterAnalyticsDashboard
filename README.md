# 🐦 Twitter Analytics Dashboard – Power BI

This project is a **Real-Time Twitter Analytics Dashboard** created in Power BI as part of the **NullClass Internship**. It includes multiple visualizations and time-based logic to analyze Twitter data effectively.

---

## 📊 **Features & Tasks**

### ✅ Task 1: Proportion of Total Clicks
- Pie chart showing the proportion of URL clicks, profile clicks, and hashtag clicks for tweets with more than 500 impressions.
- Includes a drill-down to view click types for each tweet.

### ✅ Task 2: Top 10% Tweets by Engagement Rate
- Displays tweets with highest engagement rates.
- Filters applied:
  - >50 likes
  - Posted on weekdays
  - Tweet character count <30
  - Time-based visibility: 3PM–5PM IST

### ✅ Task 9: Media Views vs Media Engagements
- Dual-axis chart showing Media Views (columns) and Media Engagements (line) by Day of Week.
- Filters applied:
  - Q4 (Oct–Dec) tweets
  - Odd dates
  - Even impressions
  - Tweet character count >30
  - Tweets without letter 'H'
- Time-based visibility: 3PM–5PM and 7AM–11AM IST.

### ✅ Task 10: App Opens vs No App Opens Comparison
- Clustered bar chart comparing engagement rates for tweets with and without app opens.
- Filters applied:
  - Weekday tweets between 9AM–5PM
  - Odd dates
  - Even impressions
  - Tweet character count >30
  - Tweets without letter 'D'
- Time-based visibility: 12PM–6PM and 7AM–11AM IST.
